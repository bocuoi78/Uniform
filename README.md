# Uniform VKU - Hệ thống hỗ trợ bán áo đồng phục VKU
## Tính năng
- Hiển thị toàn bộ danh sách đã đăng ký ("/list")
- Thêm mới đăng ký ("/add")
- Chỉnh sửa đăng ký ("/edit", id)
- Quét mã QR kiểm tra thông tin ("/")
- Xuất file excel ("/export")
## Yêu cầu
- JDK 17
- MySQL Server
